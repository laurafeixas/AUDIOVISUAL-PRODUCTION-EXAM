Una plataforma web moderna, multilingüe i interactiva per estudiar els conceptes clau de **Producció Audiovisual**.

**Disponible en:** 🇪🇸 Català | 🇬🇧 English

## 🚀 Com Usar

### 🌍 Canviar d'Idioma

1. **A la part superior dreta** hi ha els botons de llenguatge
2. Clica **Català** o **English** per canviar
3. Tot el contingut es canviarà automàticament (temari, flashcards, test)

### 📚 Estudiar el Temari Complet

1. Vés a la secció **📚 Temari**
2. Selecciona un dels 6 apartats:
   - **Equips & Rols**: Directors, càmera, so, grip...
   - **Preproducció**: Calendario, localitzacions, timings
   - **Documents AD**: Escaleta, callsheet, claqueta
   - **Script Breakdown**: Elements del guió, octaus
   - **Production Folder**: Documentació completa
   - **Pressupost**: Categories, timings, finançament
3. Llegeix les explicacions detallades
4. Pren notes mentre estudies

### 📇 Estudiar amb Flashcards

1. Vés a la secció **📇 Flashcards**
2. Clica les targetes per revelar les respostes
3. Utilitza la **barra de cerca** per buscar temes específics
4. Filtra per categoria amb els botons

### 🧪 Fer el Test

1. Vés a la secció **🧪 Test**
2. Selecciona la dificultat que vulguis
3. Respon les preguntes i veu immediatament si són correctes
4. Llegeix les explicacions per entendre millor

### 📝 Prendre Notes

1. Vés a la secció **📝 Notes**
2. Escriu les teves notes
3. Clica "Guardar" per emmagatzemar-les
4. Les notes es guarden automàticament al teu navegador

### ⚙️ Personalitzar Preferències

1. Vés a **Preferències**
2. Activa/desactiva mode fosc, so i altre opcions
3. Veu el temps que portes estudiant aquesta sessió

## 🐛 Resolució de Problemes

### Les meves dades no es guarden

- Comprova que el navegador permeti Local Storage
- Desactiva l'incògnit/navegació privada

### Els sons no funcionen

- Comprova els paràmetres d'àudio del teu dispositiu
- Assegura't que tens permís d'audio al navegador

### La web va lenta

- Neteja la memòria cau del navegador
- Tanca altres tubs/aplicacions

## 📞 Suport

Si detectes errors o tens suggeriments de millora, comprova:

1. Que uses la versió més recent del navegador
2. Que els fitxers HTML, CSS i JS estiguin a la mateixa carpeta
3. Que obres `study.html` (no els altres fitxers)

## ⏱️ Sessió de Prova

Per a una **primera sessió optima d'estudi**, segueix aquest ordre:

1. **10 min**: Llegeix els apartats del **Temari** (comença per "Equips & Rols")
2. **15 min**: Fes **Flashcards** dels conceptes que acabes de llegir
3. **10 min**: Fes el **Test** per veure si ho has après
4. **5 min**: Pren **Notes** dels punts que t'han costat més

Total: ~40 minuts per sessió.
Repeteix cada dia durant 1-2 setmanes per dominar tot el temari.
